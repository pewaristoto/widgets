Pewaris Widgets

Embed live football scores, league tables, match predictions, and togel results on any website. Free, no account required.

Getting started

Place a div where you want the widget, then include the script tag once per page. One script handles any number of widgets.

<div class="whistle-widget"
     data-type="standings"
     data-league="39"></div>

<script src="https://widgets.pewaris.dev/api/widget/script" async></script>

For togel, specify the market with data-market:

<div class="whistle-widget"
     data-type="result"
     data-market="macau"></div>

<script src="https://widgets.pewaris.dev/api/widget/script" async></script>

Embed methods

Live script — recommended

Data updates automatically. One script tag per page works for any number of widgets. Best for live scores, current standings, and daily draw results.

<div class="whistle-widget" data-type="standings" data-league="39"></div>
<div class="whistle-widget" data-type="result" data-market="macau"></div>

<script src="https://widgets.pewaris.dev/api/widget/script" async></script>

Static HTML — no JavaScript

Use the Widget Builder to export a static HTML snapshot. Plain HTML, readable by search engines, styled via a shared stylesheet. Does not update automatically — best for content that rarely changes.

<link rel="stylesheet" href="https://widgets.pewaris.dev/api/widget/style.css">
<div class="pw-w ..."> ... widget content ... </div>
One stylesheet link covers multiple static widgets on the same page.

Examples

Multiple widgets on one page:

<div class="whistle-widget" data-type="standings" data-league="39"></div>
<div class="whistle-widget" data-type="result" data-market="macau"></div>
<div class="whistle-widget" data-type="prediction" data-league="1" data-season="2026"></div>

<script src="https://widgets.pewaris.dev/api/widget/script" async></script>

Dark mode with a custom accent color:

<div class="whistle-widget"
     data-type="result"
     data-market="hkpools"
     data-theme="dark"
     data-accent="#f59e0b"></div>

<script src="https://widgets.pewaris.dev/api/widget/script" async></script>

Multi-market togel board:

<div class="whistle-widget"
     data-type="board"
     data-markets="macau,sg4d,hkpools,sdypools,sgtoto"
     data-layout="full"></div>

<script src="https://widgets.pewaris.dev/api/widget/script" async></script>
